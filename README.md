# 🐾 Pet Love

A modern web application for discovering and managing pet adoption listings.

## ✨ Features

- Browse pet listings (buy, sell, lost, found)
- User authentication with JWT tokens
- Advanced filtering & search
- User profiles & favorites
- News feed
- Friends catalog (breeders & service providers)
- Responsive design

## 🛠️ Tech Stack

- **React** 19.2.4 | **TypeScript** 5.9.3
- **Redux Toolkit** 2.11.2 | **Redux Persist** 6.0.0
- **Tailwind CSS** 4.2.2 | **Vite** 8.0.1
- **React Router** 7.13.1 | **Axios** 1.13.6
- **Formik** 2.4.9 | **Yup** 1.7.1

## 🚀 Quick Start

```bash
npm install
npm run dev       # Start dev server at localhost:5173
npm run build     # Production build
npm run lint      # Check code quality
```
